# Pok√©dolar Twitter bot

Tweets the current exchange rate from BRL to USD

Usage
 - Install requirements
    - `pip3 install requirements.txt`
 - Setup .env
    - `cp .env.example .env`
    - Add your keys to .env
 - Add each pokemon image to a folder
    - `mkdir pokemon_img`
 - Run update_bot.py
    - `python3 update_bot.py`
